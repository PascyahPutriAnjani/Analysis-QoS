## Metrik QoS
● Throughput  : Data yang berhasil dikirim.                               
● Jitter      : Kestabilan waktu mengirim paket.                        
● Packet loss : Persentase paket yang hilang.                                
● Delay       : Waktu yang dibutuhkan untuk mengirim paket.
